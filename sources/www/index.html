Getting Started

Your web application is now ready! You can start building your website by uploading your files.

Quick Start Options:

  • SFTP Access - Recommended for easy file management
  • SSH/SCP - Direct server access for advanced users
  • File Upload - Use any FTP client of your choice

SFTP Connection Details

Use these credentials to connect via SFTP and upload your website files:

Domain
__DOMAIN__
Port
__SSH_PORT__
Username
__ID__
Password
Your installation password

Password Information

Important: If you didn't provide a password during installation, a secure random password was automatically generated for you. This password is displayed above and stored in your application settings.

The generated password is cryptographically secure and 20 characters long. You can change it anytime through the YunoHost configuration panel.

Recommended SFTP Client: FileZilla (Free, cross-platform)

File Structure

Upload your website files to the following directory:

/var/www/__APP__/www/

This directory contains all the files that will be served by your web application.
